# Art : projets stéganographiques

## Idées

### 1. Archive cachée (usage immédiat)
Images de production Goldberg ou personnelles qui contiennent un texte chiffré dans leurs pixels.
Le sens de l'image et le secret coexistent. La beauté de l'image est la couverture.

### 2. Texte qui contient du texte
Un communiqué de presse, une légende, une note de dossier, qui porte en elle, invisible,
la version brute, non filtrée, de ce qui a été pensé mais pas dit.

### 3. Correspondance stéganographique
Envoyer des images "neutres" (photos de rue, frames de film)
sur des canaux publics (email, Signal) avec des messages encodés dedans.
Protocole : GHOST MODE (AES-256-GCM + LSB + bruit). La clé partagée est hors-bande.

### 4. Goldberg : l'invisible dans l'image
Images générées par le pipeline Goldberg Latent Stream qui contiennent
les posts de Joshua encodés. L'image montre une chose, porte en elle une autre.
Chaque frame = deux temporalités.

### 5. MATRYOSHKA comme forme
11 images emboîtées. L'image extérieure est "banale".
Chaque couche révèle quelque chose. La 11e est le secret final.
Référence : poupée russe, mais aussi structure narrative du film.

### 6. Stéganographie textuelle dans les dossiers CNC
Intégrer des messages zero-width dans les textes institutionnels.
La note d'intention contient, invisible, la vraie note d'intention.
(Artistique / conceptuel, pas pour tromper une commission.)
